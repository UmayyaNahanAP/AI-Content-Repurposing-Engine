# AI Content Repurposing Engine

## Project Overview
This project automatically transforms a single blog/article into multiple platform-specific content formats (LinkedIn, Twitter, YouTube, Newsletter) while preserving meaning and adjusting tone and style.

## Features
- Parse blog content from URL or raw text
- Extract key insights using semantic analysis
- Generate LinkedIn posts, Twitter threads, YouTube scripts, and newsletter summaries
- Quality review agent ensures readability, tone, and factual consistency
- Download outputs in text/Markdown format
